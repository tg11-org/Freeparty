Changelog

A running log of major updates and improvements to Freeparty.

2026-04-19 — Policy & Support Expansion

  • Expanded Terms of Service with age/jurisdiction table (US COPPA, EU GDPR, UK, Canada, Australia).
  • Expanded Privacy Policy with cookies, data retention categories, user rights, and cross-border processing notes.
  • Expanded Community Guidelines with explicit moderation enforcement flow (warnings → restrictions → suspension → ban).
  • Expanded FAQ with practical answers: password reset, account deletion, block vs mute, DM keys.
  • Added About page trust & safety section.
  • Added Support page with intake form that generates a pre-filled mailto: link to support@tg11.org.
  • Added effective date (2026-04-19) and version metadata to all policy pages.

2026-04-19 — Footer & Public Pages

  • Added global site footer with navigation links: About, Terms, Privacy, Guidelines, FAQ, Support, Status, Changelog.
  • Added Copyleft notice and GitHub repository link to footer.
  • Created six public pages: About, Terms of Service, Privacy Policy, Community Guidelines, FAQ, Support.

2026-04-19 — Direct Messages: Security & Keys Menu

  • Added collapsible Security & Keys panel (☰) to the DM conversation view.
  • Fingerprint display, device/key inventory, and Browser E2E Mode controls are all grouped inside the collapsible menu.
  • Sticky compose area is now clean — textarea and send only, no key management clutter.

2026-04-19 — Profile Settings: Toggle Switch UI

  • Replaced raw checkbox inputs on profile edit page with accessible CSS toggle switches.
  • Toggles cover: show follower count, show following count, private account, auto-reveal spoilers.
  • Global toggle and checkbox/radio styles applied across all forms.

2026-04-19 — Profile Links (Linktree-style)

  • Added ProfileLink model: title, URL, display order, active toggle, profile association.
  • Users can manage custom links at My Links.
  • Public links page available at /profiles/<handle>/links/.
